# Custom Domain Setup Guide

## Get Your Own Short Link (like fincke.dev)

### Option 1: Get a .dev Domain (Recommended - Like fincke.dev)

1. **Purchase a .dev domain:**
   - Go to [Google Domains](https://domains.google/) or [Namecheap](https://www.namecheap.com/)
   - Search for your desired domain (e.g., `yourname.dev`)
   - Purchase the domain (usually $12-20/year)

2. **Add Domain to Vercel:**
   - Go to: https://vercel.com/jeevesh-srivastavas-projects-2eae2842/my-portfolio-jk1806/settings/domains
   - Click "Add Domain"
   - Enter your domain (e.g., `yourname.dev`)
   - Follow the DNS configuration instructions

3. **Configure DNS:**
   - Vercel will provide DNS records to add
   - Add them in your domain registrar's DNS settings
   - Wait for DNS propagation (usually 5-30 minutes)

### Option 2: Use Vercel's Free Subdomain

Vercel automatically provides a shorter URL. Your current URL is:
- `https://my-portfolio-jk1806-9fpr0ek6i.vercel.app`

You can request a cleaner subdomain in Vercel settings.

### Option 3: Use a Link Shortener

For a very short link, you can use:
- [bit.ly](https://bit.ly) - Create a short link
- [tinyurl.com](https://tinyurl.com) - Free URL shortener
- [rebrandly.com](https://rebrandly.com) - Custom branded short links
